참석자: 황일순 대표 (보트랜스), 승근수 대표 (태경홀딩스), 센터장 (의왕/서이천), 김지훈 (개발자)
장소: 의왕센터 또는 서이천센터 (미정)

사전 준비 (시연 전날까지)

항목내용
시연 센터 확정의왕 or 서이천 → Supabase sites 테이블에 해당 센터 GPS 좌표 정확히 등록
테스트 단말스마트폰 2~3대 (APK 설치) — 김지훈 + 센터장 + 여분 1대
계정 세팅시연용 근로자 계정 3~4명 (지게차, 피커, 검수 등 파트별), 센터장 계정 1명
노트북관리자웹 시연용 (크롬 풀스크린)
캘린더 일정시연 당일 "MVP 시연" 일정 미리 등록 (캘린더 기능 자연스럽게 노출)
더미 데이터지난 1주일치 출퇴근 기록 + 급여 데이터 입력 (화면이 풍성해 보임)
인터넷현장 Wi-Fi 확인 + 모바일 핫스팟 백업 준비

시연 시나리오

1 출근 시연 (5분) — 현장감

"실제 현장에서 근로자가 출근하는 과정을 보여드리겠습니다"

순서행동포인트
1센터장이 Worker App으로 SMS 인증 로그인"전화번호만으로 간편 로그인"
2김지훈이 다른 폰으로 근로자 계정 로그인동시 2명 출근 시연
3출근 버튼 터치 → GPS 검증 → 성공"센터 반경 내에서만 출근 가능, 대리출근 불가"
4센터 밖에서 출근 시도 → 실패 보여주기GPS 검증 신뢰성 (사전에 밖에서 한번 찍어두기)
핵심 멘트: "기존 수기/엑셀 관리 대비 GPS로 위치 검증되니 정확한 출근 기록이 자동 생성됩니다"

2 관리자웹 실시간 확인 (5분) — 대표님들에게

"방금 출근한 기록이 관리자 화면에 바로 반영됩니다"

순서행동포인트
1노트북으로 관리자웹 대시보드 오픈캘린더에 오늘 일정, 요약카드에 출근 인원수 반영
2출퇴근 현황 테이블에 방금 출근한 2명 표시"실시간 모니터링 가능"
3이름 클릭 → 근로자 상세 페이지"개인별 정보 한눈에 확인"
4센터 필터 드롭다운 전환"센터별 필터링, 전체 현황 파악"

3 조퇴 시연 (3분)

순서행동포인트
1Worker App에서 조퇴 버튼 → 사유 입력 ("병원 방문")"사유가 기록으로 남습니다"
2관리자웹 새로고침 → 상태 "조퇴" 뱃지 + 비고에 사유실시간 반영 확인

4 퇴근 시연 (2분)

순서행동포인트
1나머지 계정 퇴근 버튼 터치GPS 검증 후 퇴근 처리
2관리자웹에서 근무시간 자동 계산 확인"출퇴근 시간 기반 자동 계산"

5 관리 기능 데모 (7분) — 핵심 가치

"축적된 데이터로 할 수 있는 것들을 보여드리겠습니다"

순서행동포인트
1근태기록 탭 → 날짜 범위 조회"원하는 기간별 조회 가능"
2근태 수정/삭제 (관리자 권한)"실수나 예외 상황 관리자가 보정 가능"
3급여관리 탭 → 급여 목록 확인"근태 데이터 기반 급여 자동 산출"
4급여 확정 → 엑셀 다운로드"확정 후 엑셀로 바로 추출, 기존 업무 호환"
5설정 → 공지사항 등록"근로자 앱에 공지사항 실시간 전달"
6계정관리 → 새 관리자 계정 생성"센터장/대표 계정 바로 발급 가능"
TIP: 엑셀 다운로드는 대표님들이 가장 관심 가지는 기능입니다. "기존 엑셀 업무와 100% 호환"을 강조하세요.

시연 팁

1. GPS 실패 시연을 꼭 포함하세요
"센터 밖에서는 출근 불가"가 가장 강력한 셀링포인트입니다. 사전에 센터 밖에서 실패 화면을 스크린샷으로 준비해두면 현장에서 자연스럽게 보여줄 수 있습니다.
2. 더미 데이터를 충분히 넣으세요
지난 1주일치 출퇴근 + 급여 데이터가 있어야 근태기록/급여 탭이 빈 화면이 아닌 실제 운영처럼 보입니다.
3. 시연 순서는 반드시 "현장→관리자웹" 순으로
현장에서 직접 출근 찍고 → 노트북에서 바로 확인하는 흐름이 "실시간"의 임팩트를 줍니다.
4. 인터넷 백업 필수
물류센터 Wi-Fi가 불안정할 수 있습니다. 모바일 핫스팟을 반드시 준비하세요.

시연 전 체크리스트

  • 시연 센터 GPS 좌표 등록 확인 (Supabase sites 테이블)
  • APK 빌드 + 테스트 단말 2~3대 설치
  • 시연용 근로자 계정 3~4명 생성 (파트별)
  • 센터장 계정 생성 + 로그인 테스트 완료
  • 1주일치 더미 출퇴근/급여 데이터 입력
  • 캘린더에 시연 당일 일정 등록
  • 공지사항 1~2건 등록
  • 관리자웹 로그인 테스트 (시스템관리자 + 데모 로그인)
  • 현장 인터넷 환경 확인 (Wi-Fi + 핫스팟 백업)
  • GPS 실패 화면 스크린샷 백업 준비
