# 카카오 개발자 플랫폼 운영정책 요약

> 원본: https://developers.kakao.com/terms/latest/ko/site-policies
> 확인일: 2026-02-16

---

## 제3조. 서비스 이용이 제한되는 카테고리

다음 서비스는 이용이 제한됩니다:
- 게임 애플리케이션 및 웹
- 게임 관련 애플리케이션/웹 (사전예약, 리워드 등)
- 가상자산 관련 서비스 (거래, 보관, 예치 등)

---

## 제4조. 금지된 내용

애플리케이션에서 표시되는 내용으로 금지되는 항목:

### 법률 위반 관련
- 제3자 권리(저작권 등) 침해 내용
- 의료법, 학원법, 대부업법 등 현행법 위반 내용
- 무허가 주식리딩, 상담 서비스, 카피 트레이딩, AI 자동매매 등 미인가 금융투자상품 정보

### 청소년 보호
- 성인방송, 성인커뮤니티, 성인용품, 만남주선 등 (단, Kakao 인증서 나이확인 시 예외 가능)

### 사회질서 침해
- 지나친 세속적/모독적 표현
- 증오 및 폭력적 내용
- 인종/민족 차별 옹호
- 컴퓨터 해킹/크래킹 조장

### 불법 활동
- 사행성 서비스 (스포츠토토, 도박, 카지노, 경마, 복권 등)
- 담배/주류/총포도검 (단, 성인인증 제공 시 예외 검토 가능)
- 규제 약물, 불법 소프트웨어
- 마약 관련 도구
- 피싱, 악성 콘텐츠
- 무한 팝업 등 사용자에게 불편 초래하는 행위

---

## 제5조. 금지된 행동

### 스팸 관련
- 원치 않는 상업용 이메일 생성
- 타인 모방, 허위 계정 생성
- 데이터 마이닝으로 이메일 수집
- 메일링 리스트 삭제자에게 이메일 발송

### 불법 콘텐츠 전송
- 불법, 명예훼손, 희롱, 사기, 외설 내용 배포
- 바이러스, 트로이 목마 등 악성 파일 배포
- 미성년자 유해 콘텐츠 전송

### 부정행위
- 타인 가장, 신원 사칭
- 지적재산권 무단 전송
- 사용자 동의 없이 사용자를 대신해 콘텐츠 게시
- 타인 카카오톡 서비스 이용 방해
- 허가되지 않은 상업적 재판매

### 기술 악용
- 리버스 엔지니어링
- 저작권/상표 고지 삭제
- 불법 P2P 파일 공유

### 정보 부정행위
- 프로필, 앱 정보 허위 입력
- 정보 변경 후 미수정, 단축 도메인 악용 등
- 개발자 계정 정보 거짓 기재

### 다중 계정 운영
- 동일 서비스에서 여러 개발자 계정 및 앱 생성 운영

### 정보 무단 이용
- 데이터, 비밀 키, 액세스 토큰 등 무단 복사/복제/변경/배포

## 제6조. 쿼터 및 제한

| 항목 | 제한 | 유형 |
|------|------|------|
| 요청 길이 | 30초 이내 | Hard |
| MAU | 50만명 초과 제한 | Hard |
| 일일 API 호출 | 1,000만 건 초과 제한 | Hard |
| 네트워크 대역폭 | 2TB/월 | Soft |

## 제8조. 이용제한

위반 시 단계적 제한: 경고 → 데이터 삭제 → 일시정지 → 계약해지

제3조, 제4조, 제5조 위반 또는 법령 위반 시 즉시 계약 해지 가능
