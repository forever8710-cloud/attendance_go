# v1.15 — Worker App 센터 선택 UI 개선

## 변경 사항

### 프로필 입력 화면 센터 선택
- 프로필 완성(profile_completion_screen) 화면에서 센터 선택 드롭다운 추가
- 센터 목록: 서이천(IC), 안성(AS), 의왕(UW), 부평(BP)
- auth_repository에 센터 정보 저장 로직 연동
- auth_provider에서 프로필 미완성 시 센터 선택 화면으로 전환

## 변경 파일
- `apps/worker_app/lib/features/auth/presentation/profile_completion_screen.dart`
- `apps/worker_app/lib/features/auth/data/auth_repository.dart`
- `apps/worker_app/lib/features/auth/providers/auth_provider.dart`
