# v1.16 — 앱 이름 변경 + 개인정보 동의 화면 추가

## 변경 사항

### 1. 앱 이름 변경
- "출퇴근GO" → "WorkFlow" + "by tkholdings" 서브텍스트
- 로그인 화면 타이틀, web/index.html, web/manifest.json 모두 반영

### 2. 개인정보 동의 화면 추가
- 프로필 입력 전 별도 화면으로 개인정보 수집·이용 동의 폼 추가
- **[필수] 개인정보 수집·이용 동의**: 이름, 전화번호, 주민등록번호, 주소, 계좌정보 수집
- **[선택] 위치정보 이용 동의**: GPS 위치정보 수집
- 전체 동의 체크박스로 한번에 토글 가능
- 필수 미체크 시 "동의하고 계속하기" 버튼 비활성화
- 동의 후 추가정보 입력(프로필) 화면으로 이동

### 3. AuthStatus 흐름 변경
- `needsConsent` 상태 추가 (needsPhoneVerification → needsConsent → needsProfileCompletion)
- 카카오 로그인 / 구글→전화인증 후 프로필 미완성 시 `needsConsent`로 전환
- `acceptConsent()` 메서드로 동의 완료 → `needsProfileCompletion`으로 전환
