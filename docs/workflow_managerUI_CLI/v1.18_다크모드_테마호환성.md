# v1.18: Manager Web 다크모드 테마 호환성 완성

## 작업 일자
2026-02-13

## 변경 내용

### 다크모드 테마 지원
- 전체 화면의 하드코딩 색상(`Colors.white`, `Colors.grey[200]!`, `Colors.indigo` 등)을 `Theme.of(context).colorScheme` 기반으로 전환
- 다크 테마 커스텀 색상 설정:
  - `surface`: `#1E1E2E`
  - `scaffoldBackground`: `#161624`
  - `cardColor`: `#252538`
  - `dividerColor`: `#3A3A50`

### 수정된 파일 (10개)
1. `main.dart` — 다크 테마 색상, 상단바, 푸터, 콘텐츠 영역
2. `side_nav_drawer.dart` — 사이드 네비 배경, 아이콘, 텍스트
3. `login_screen.dart` — 로그인 배경, 텍스트
4. `sticky_data_table.dart` — 테이블 헤더, 행, 테두리
5. `dashboard_screen.dart` — 요약 카드, 검색, 드롭다운
6. `workers_screen.dart` — 근로자 카드, 폼 필드, 드롭다운
7. `attendance_records_screen.dart` — 근태기록 카드, 검색
8. `payroll_screen.dart` — 급여 드롭다운, 빈 상태
9. `settings_screen.dart` — 설정 탭, 테마 카드, 스위치
10. `worker_detail_screen.dart` — 상세 카드, 정보 필드, 요약 카드

### 적용 패턴
- `cs.surface` → 배경색 (기존 `Colors.white`)
- `cs.onSurface.withValues(alpha: x)` → 텍스트/아이콘 (기존 `Colors.grey[x]`)
- `cs.outlineVariant.withValues(alpha: 0.3)` → 테두리 (기존 `Colors.grey[200]!`)
- `cs.primary` → 강조색 (기존 `Colors.indigo`)
- `cs.surfaceContainerHighest.withValues(alpha: 0.3)` → 입력 필드 배경
