# v1.34 Color Scheme + T.H Brand Icon Update

**작업일:** 2026-02-24

## 변경 요약

Teal 기반 색상 체계를 Lavender Grey / Space Indigo 조합으로 전면 교체하고, 방패(shield) 아이콘을 토스 스타일 원형 "T.H" 이니셜 아이콘으로 교체.
