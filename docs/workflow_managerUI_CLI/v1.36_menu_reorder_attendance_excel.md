## 근태현황 날짜 프리셋 칩

- `ChoiceChip` 위젯 사용
- 프리셋: 오늘 / 이번주 / 이번달 / 전체기간(2024-01-01~오늘)
- 선택 시: Space Indigo 배경 + 흰색 텍스트
- 비선택 시: 테마 outlineVariant 테두리
- 수동 캘린더 선택 시 프리셋 해제 (`_datePreset = ''`)

## 근태현황 엑셀 내보내기

- `AttendanceExcelExport.exportToExcel(data, dateRangeLabel)`
- 파일명: `근태현황_시작일_종료일.xlsx`
- 컬럼: No., 날짜, 성명, 직위, 직무, 사업장, 출근, 퇴근, 근무시간, 상태
- 헤더: Bold + 연보라 배경 (#E8EAF6)
- 합계 행: Bold
- 브라우저 Blob 다운로드 (`package:web`)
