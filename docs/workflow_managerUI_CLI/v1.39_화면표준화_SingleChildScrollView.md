# v1.39 — 전체 화면 레이아웃 표준화 (SingleChildScrollView + SizedBox 패턴)

**작업일**: 2026-02-26

---

## 배경

- 승근수 대표 노트북에서 테이블이 잘려 보이는 문제 (모니터 크기/해상도별 레이아웃 차이)
- 기존 `Expanded` 기반 레이아웃은 뷰포트 크기에 따라 테이블 영역이 축소됨
- **목표**: 모든 모니터에서 동일한 화면 구조로 표시 (표준화)

## 표준 패턴

```dart
// 모든 화면에 적용하는 표준 레이아웃
return SingleChildScrollView(       // ← 전체 페이지 스크롤
  child: Column(
    children: [
      Padding(/* 상단 컨트롤 영역 */),
      SizedBox(                     // ← 테이블 고정 높이
        height: 500,
        child: StickyHeaderTable.wrapWithCard(...),
      ),
    ],
  ),
);
```

**핵심 규칙**:
- `Expanded` 사용 금지 (뷰포트 크기에 종속)
- 테이블 영역: `SizedBox(height: 500)` 고정
- 전체를 `SingleChildScrollView`로 래핑 → 항상 스크롤 가능
- `StickyHeaderTable` 내부 `ListView`는 `SizedBox(500)` 안에서 자체 스크롤
