## 레이아웃 변경: 좌우 마스터-디테일 → 상하 분할

### 근로자 관리 (`workers_screen.dart`)

- [x] **상단**: 인사기록카드 Card
  - 카드 헤더 우측에 "신규 등록" 버튼 배치
  - 좌측: 사진폼 단독 배치 (110x140, 반명함 + 사진첨부 버튼)
  - 우측: 입력 폼 (ID/사번/이름/주민번호/성별 등 5행)
- [x] **중간**: 구분선 레이아웃 (아이콘 + "근로자 목록" 타이틀 + Divider + 검색창)
- [x] **하단**: DataTable 근로자 리스트 (사번 컬럼 추가, 클릭 시 상단 폼에 로드)

### 계정 관리 (`accounts_screen.dart`)

- [x] **상단**: 계정 정보 Card
  - 카드 헤더 우측에 "신규 등록" 버튼 배치
  - 1행: ID(읽기전용) | 이름 | 전화번호 | 이메일
  - 2행: 역할 드롭다운 | 상태 ChoiceChip | 생성일
- [x] **중간**: 구분선 레이아웃 (아이콘 + "계정 목록" 타이틀 + Divider + 검색창)
- [x] **하단**: DataTable 계정 리스트 (이름/전화/이메일/역할/상태/생성일)
